# Xentic Comply API - Integration Examples

Welcome to the Xentic Comply API examples repository! 
Here you'll find ready-to-use code snippets in various languages to integrate automated compliance scanning into your apps, dashboards, or CI/CD pipelines.

[➡️ Get your API Key on RapidAPI](https://rapidapi.com/xentic-xentic-default/api/xentic-comply-api)

## 📂 Directory Structure

* `/curl`: Simple bash scripts to test the API via command line.
* `/javascript`: Node.js examples using `axios` (great for React/Next.js/Node backends).
* `/python`: Python examples using `requests`, including a bulk scanner script.

## 🚀 Quick Start

1. Subscribe to the API on RapidAPI (the **BASIC plan is free!**).
2. Copy your `x-rapidapi-key`.
3. Pick your preferred language folder and run the scripts!

## 🔐 Security Note
Never hardcode your RapidAPI key in frontend code (like React or Vanilla JS running in a client's browser). Always call the RapidAPI endpoints from your backend servers to protect your quota and billing.
